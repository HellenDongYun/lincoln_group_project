Tips for Volunteers
  • Identification: Use either User ID number OR email address to identify participants
  • User ID: Ask participants for their system user ID (from their profile)
  • Email: More practical - ask for their registered email address
  • Completion Time: Enter the actual time participant finished (not current time)
  • Time Format: Use HH:MM:SS format (e.g., 14:35:20 for 2:35:20 PM)
  • Validation: System will verify participant is registered for the selected event
  • Duplicates: If a time already exists, it will be updated with the new time
